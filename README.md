# watch_together

A project can watch video together on different device.
利用dlna 投屏原理实现多端 异地一起看视频的项目；
flutter 编写，支持多平台；
目前支持 android，windows，iphone，ipad，linux；
已测试：android，iphone，Windows；

项目开源，不上架应用市场，安卓用户和windows 用户可以点击下面链接体验；
